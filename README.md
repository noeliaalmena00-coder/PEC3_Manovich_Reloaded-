## Introducción

> "El software se ha convertido en el principal motor de la cultura contemporánea” (Manovich, 2013)[^1]

Está idea es la clave para analizar cómo las prácticas culturales de hoy en día están profundamente condicionadas por los sistemas digitales. Desde esta perspectiva, algunas plataformas y productos culturales como pueden ser TikTok o Just Dance, no pueden entenderse únicamente como medios de entretenimiento. Deben de tenerse en cuenta como ejemplos de hibridación cultural mediados por el software.


## Re-descubriendo la hibridación con TikTok
![enter image description here](https://www.panoramaaudiovisual.com/wp-content/uploads/2022/05/TikTok-Audiovisual-Broadcast-Guia.jpg)

TikTok constituye uno de los ejemplos más representativos de la cultura del software actual y es un caso paradigmático de hibridación avanzada que encaja de manera perfecta con los planteamientos que nos presenta Lev Manovich en  _El software toma el mando_. A diferencia de otras plataformas audiovisuales que ya existían, tal y como menciona Manovich, TikTok no solo se limita a distribuir el contenido, también lo organiza y lo prioriza de forma constante mediante algoritmos, convirtiendo el software en un elemento central del proceso cultural[^2].

Desde el punto de vista de la hibridación, TikTok se dedica a fusionar diferentes medios que ya habían existido de forma individual: los vídeos cortos, la música y los audios populares, el texto superpuesto, la animación gráfica, la edición audiovisual del contenido y la interacción social entre los usuarios. Sin embargo, tal como señala Lev Manovich, lo más importante no es la suma de todos estos medios, sino la transformación que se produce cuando estos se integran bajo una lógica computacional común[^3]. En TikTok, todos estos elementos dependen del software y de su algoritmo. 

Uno de los conceptos de Manovich que está muy presente en TikTok es la oposición entre narrativa y base de datos[^4] La plataforma no cuenta historias de forma lineal, sino que tiene una colección casi infinita de vídeos organizados como una base de datos dinámica. El usuario navega por una secuencia personalizada de contenidos generada por el sistema en la que el algoritmo actúa como si fuese un narrador invisible.

En TikTok la creatividad también está de cierta forma condicionada por el software. Las herramientas de edición integradas, los filtros, los efectos y, sobretodo, los sonidos reutilizables, fomentan una cultura del remix y la repetición. Este sistema modular encaja con el principio de modularidad descrito por Manovich, ya que cada vídeo se crea a partir de piezas que pueden recombinarse de manera constante.

Otro aspecto clave es el cambio en el papel que tiene el usuario. En TikTok, este no solo consume o crea contenidos, también va generando datos con cada interacción que realiza. Estas interacciones "alimentan" el sistema, que sigue adaptando de manera única la experiencia de cada usuario, reforzando la idea de Manovich de que el software es un agente activo.

En este contexto, TikTok representa una evolución extrema de los nuevos medios estudiados y analizados por Manovich. El software no solo "toma el mando", también influye directamente en cómo se crean, circulan y consumen. Por este motivo, TikTok sería un caso clave en una versión actualizada de  _El software toma el mando_, al representar la hibridación perfecta entre cultura, algoritmo y participación masiva.
